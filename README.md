## 1. 产品概述

### 1.1 产品定位

**SoundNote** 是一款 Android 原生录音应用，主打简洁高效的录音体验，支持录音、查询和管理功能。

### 1.2 目标用户

- 需要随时记录会议、讲座、采访的用户
- 需要录制笔记、备忘、语音日记的用户
- 对录音品质有不同需求的 professionals

### 1.3 核心价值

- **简单易用**：一键录音，无需复杂设置
- **多格式支持**：支持 MP3、AAC、M4A、WAV 等主流音频格式
- **灵活品质**：从低品质省空间到超高保真，满足不同场景需求
- **本地存储**：录音数据安全存储在设备本地

#### 2.1.2 录音格式

| 格式 | 扩展名 | MIME类型 | 说明 |
|------|--------|---------|------|
| MP3 | .mp3 | audio/mpeg | 通用格式，兼容性好 |
| AAC | .aac | audio/aac | 高效压缩 |
| M4A | .m4a | audio/mp4 | Apple生态兼容 |
| WAV | .wav | audio/wav | 无损格式 |

#### 2.1.3 录音品质

| 品质等级 | 显示名 | 采样率 | 比特率 | 声道 | 适用场景 |
|---------|--------|--------|--------|------|---------|
| 低品质 | LOW | 22050 Hz | 64 kbps | 单声道 | 长时间录音、语音备忘 |
| 标准品质 | STANDARD | 44100 Hz | 128 kbps | 单声道 | 日常录音 |
| 高品质 | HIGH | 44100 Hz | 192 kbps | 双声道 | 会议、访谈 |
| 超高品质 | ULTRA | 48000 Hz | 320 kbps | 双声道 | 音乐录制、专业场景 |

## 3. 数据结构

### 3.1 数据库设计

#### 3.1.1 录音表 (recordings)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | String (PK) | UUID 唯一标识 |
| fileName | String | 文件名 |
| filePath | String | 文件路径 |
| format | String | 音频格式 |
| durationMs | Long | 时长（毫秒） |
| fileSizeBytes | Long | 文件大小（字节） |
| sampleRate | Int | 采样率 |
| bitRate | Int | 比特率 |
| channels | Int | 声道数 |
| isFavorite | Boolean | 是否收藏 |
| folderId | String? | 所属文件夹 |
| transcription | String? | 转录文本（预留） |
| createdAt | Long | 创建时间戳 |
| updatedAt | Long | 更新时间戳 |

#### 3.1.2 标记表 (markers)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | String (PK) | UUID 唯一标识 |
| recordingId | String (FK) | 关联录音ID |
| timestampMs | Long | 时间戳位置（毫秒） |
| note | String? | 标记备注 |
| createdAt | Long | 创建时间戳 |

#### 3.1.3 标签表 (tags)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | String (PK) | UUID 唯一标识 |
| name | String | 标签名称 |
| color | String | 标签颜色 |
| sortOrder | Int | 排序顺序 |
| createdAt | Long | 创建时间戳 |

#### 3.1.4 文件夹表 (folders)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | String (PK) | UUID 唯一标识 |
| name | String | 文件夹名称 |
| parentId | String? | 父文件夹ID |
| isSystem | Boolean | 是否系统文件夹 |
| sortOrder | Int | 排序顺序 |
| createdAt | Long | 创建时间戳 |

#### 3.1.5 录音-标签关联表 (recording_tags)

| 字段 | 类型 | 说明 |
|------|------|------|
| recordingId | String (PK) | 录音ID |
